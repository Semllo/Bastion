2.2.- Soluciones WAF (Web Application Firewall)

Como ya hemos visto antes, una de las herramientas principales y más importantes es el uso de un WAF, así que vamos a explicar en profundidad qué es y cómo funciona.

¿Qué es?

Un WAF, o Web Application Firewall, es como su nombre indica un firewall de aplicación el cual se encarga de la seguridad de las aplicaciones web protegiéndolo de múltiples tipos de ataque como puede ser XSS o inyecciones SQL. También se encarga de la monitorización del tráfico web como lo haría un firewall.

Trabaja en la capa 7 (la capa de aplicación) por lo que obviamente solo actúa en lo que es la propia aplicación. El WAF funciona como un proxy inverso haciendo que cuando un usuario quiera navegar por nuestra web primero tenga que pasar por este WAF.

Como cualquier otro firewall, tiene una serie de reglas por las cuales se puede gestionar cualquier tipo de conexión a estas aplicaciones.
