6.2.- Herramientas

Las herramientas más comunes son:

Low Orbit Ion Cannon (LOIC): es una aplicación que realiza pruebas de estrés mediante ataques a la capa de protocolos TCP/UDP. Para ello utiliza una interfaz de WYSIWYG (What you see is what you get).

High Orbit Ion Cannon (HOIC): tiene la misma funcionalidad que LOIC, pero con más capacidades y personalizaciones. Utiliza el protocolo HTTP para lanzar ataques y está diseñada para realizar ataques en grupo.

Slowloris: es una aplicación usada para provocar un ataque bajo y lento en el servidor que sea atacado. Lo bueno de esta aplicación es que no necesita de una gran cantidad de recursos para hacer daño.

R-U-Dead-Yet (R.U.D.Y): es una herramienta similar a Slowloris que permite ataques sencillos mediante una interfaz en la que selecciona su objetivo. Esta herramienta trata de ir abriendo solicitudes HTTP de tipo POST poco a poco, manteniéndolas abiertas hasta que el servidor colapsa.
