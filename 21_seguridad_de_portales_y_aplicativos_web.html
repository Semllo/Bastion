2.1.- Seguridad de portales y aplicativos web

¿Qué es?

La seguridad aplicada a los sitios web, es aquella seguridad que se encarga precisamente de proteger las diferentes aplicaciones y portales web en los diferentes aspectos de la misma, es decir:

  • Cuando accedemos a una web.
  • Cuando modificamos su contenido.
  • Durante la navegación y uso común de la misma.

Los sitios web deben implementar distintos tipos de seguridad, entre ellos existen distintos estándares como OWASP para una correcta protección de los mismos.

Para saber más

OWASP

Es fundación sin ánimo de lucro en forma de comunidad de desarrolladores y tecnólogos que mejoran la seguridad del software. La Fundación OWASP proporciona una plataforma para mejorar la seguridad del software, basada en distintos tipos de proyectos (93 proyectos activos), con una colección de tareas relacionadas que tienen una hoja de ruta definida para asegurar el software del mundo.

Como hemos comentado dispone de distintos proyectos entre los que destacan el OWASP Top 10 que es un documento de concienciación estándar para los desarrolladores y la seguridad de las aplicaciones web. Representa un amplio consenso sobre los riesgos de seguridad más críticos para las aplicaciones web.

Reconocido globalmente por los desarrolladores como el primer paso hacia una codificación más segura, lo que hace que las empresas adopten documentos como este para asegurar que sus aplicaciones web minimicen estos riesgos. El uso del Top 10 de OWASP es quizás el primer paso y más efectivo para cambiar la cultura de desarrollo de software dentro de una organización, iniciando el camino hacia la producción de aplicaciones con código más seguro.
